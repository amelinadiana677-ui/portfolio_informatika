Курс: Введение в Python
Полное описание проекта:

Курс предназначен для учеников 9-11 классов, которые только начинают знакомство с программированием. Включает 12 модулей, охватывающих базовые концепции Python:

  • Типы данных и переменные
  • Условные операторы и циклы
  • Функции и модули
  • Работа с файлами
  • Основы ООП

Используемые технологии: Python 3.9+, Jupyter Notebook, Git

Курс: Основы баз данных
Полное описание проекта:

Курс знакомит учеников с основами реляционных баз данных и языком SQL. Особое внимание уделяется практическим навыкам:

  • Проектирование баз данных
  • Создание таблиц и связей
  • Основные SQL-запросы (SELECT, INSERT, UPDATE, DELETE)
  • JOIN-запросы и агрегатные функции
  • Основы нормализации

Используемые технологии: MySQL, SQLite, Python (sqlite3), DB Browser
